# Basic python project 

A collection of beginner friendly python project to practice code programming concepts 

--- 

## Project lists

### 1. Simple calculator 
A command line calculator that performs basic operations

#### Features 
- Supports operations `+`, `-`, `*`, `/`, `%`
- Handle invalid inputs usin exception handling 
- Prevent crash from divided or modulo by zero 

### Getting started 

#### Clone the repository 
```bash
git clone git@github.com:susmeeta223/python-basic.git
```

#### Navigate to the project directory
```bash
cd python-basic
```

#### Run the calculator
```bash
python simple_calculator.py
```
